- Como previnir:
	Para qualquer consulta a dados, utilizar algum ORM para sanitizar os dados. Caso năo queira usar nenhuma lib para
fazer esse trabalho, utilizar o SqlCommand e passe os parametros utilizando o parameters. Em nenhuma hipotese, utilize
a concatenaçăo de strings para montar um comando sql, para năo permitir comandos como 'or 1 = 1'
